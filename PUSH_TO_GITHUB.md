# Push to GitHub - Quick Commands

Your repository is already connected to: https://github.com/Rozqy/authenticity-check-pro

## Step 1: Configure Git (if not already done)

```bash
git config --global user.name "Rozqy"
git config --global user.email "your-email@example.com"
```

## Step 2: Commit All Files

```bash
git commit -m "Initial commit: Authenticity Check Pro - Complete platform for verifying drinks, cosmetics, and fragrances"
```

## Step 3: Push to GitHub

```bash
git push -u origin main
```

If you get an error about authentication, you may need to:
- Use a Personal Access Token instead of password
- Or set up SSH keys

## Step 4: Verify on GitHub

Visit: https://github.com/Rozqy/authenticity-check-pro

You should see all your files there!

## Next: Deploy to Vercel

1. Go to https://vercel.com
2. Sign in with GitHub
3. Import repository: `Rozqy/authenticity-check-pro`
4. Add environment variables (see GITHUB_DEPLOY.md)
5. Deploy!
